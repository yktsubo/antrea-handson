# antrea-handson

To try bits and pieces of antre cni functionality.

Based on release v0.13.1

## Installaion

```
wget https://github.com/vmware-tanzu/antrea/releases/download/v0.13.1/antrea.yml
```

Edit `antrea.yaml` if you want to enable features and apply to your cluster.

```
kubectl apply -f ./antrea.yml
```
